# 🌍 Wanderlust – Airbnb Inspired Full Stack Web Application

Wanderlust is a full-stack web application inspired by Airbnb. It allows users to explore, create, and manage property listings with images, locations, and reviews. The project is built using Node.js, Express, MongoDB, and EJS with a responsive frontend using Bootstrap.

## 🛠️ Tech Stack

- Frontend: HTML, CSS, Bootstrap, EJS
- Backend: Node.js, Express.js
- Database: MongoDB Atlas
- Authentication: Passport.js
- Image Storage: Cloudinary
- Other: Map Integration, REST APIs

---

## ✨ Features

- 🔐 User Authentication (Login / Register / Logout)
- 🏠 Create, Edit, Delete Listings (CRUD)
- 🖼️ Image Upload with Cloudinary
- 📍 Location-based Listings with Maps
- 🔍 Category Filtering System
- 🔎 Search Functionality (by title, location, country)
- ⭐ Add Reviews to Listings
- 📱 Fully Responsive UI
- ⚡ Flash Messages for actions
